API Documentation

Languages:

  1. View All Languages

    Endpoint: GET http://localhost:3000/api/languages

    Request Method: GET

    Usage: curl http://localhost:3000/api/languages

  2. Create a New Language Entry

    Endpoint: POST http://localhost:3000/api/languages

    Request Method: POST

    Request Headers:

    Content-Type: application/json

    Request Body:

    {
    "name": "Language Name",
    "proficiency": "Language Proficiency"
}

    Usage: curl -X POST -H "Content-Type: application/json" -d '{"name":"Language Name","proficiency":"Language Proficiency"}' http://localhost:3000/api/languages

  3. Delete a Language Entry

    Endpoint: DELETE http://localhost:3000/api/languages/:id

    Request Method: DELETE

    Usage: curl -X DELETE http://localhost:3000/api/languages/:id
